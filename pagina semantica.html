TITULO DO ARTIGO 1

CONTEUDO DO ARTIGO

● Basicamente o CSS (Folha de Estilo em Cascata) é o código que você usa para dar estilo à sua página Web. ● Assim como o HTML, o CSS não é uma linguagem de programação. Também não é uma linguagem de marcação — é uma linguagem de folhas de estilos. ● Isso significa que o CSS permite aplicar estilos seletivamente a elementos em documentos HTML.

titutlo da barra lateral

background-color: Define a cor de fundo de um elemento HTML.
background-image: Define uma imagem como fundo do elemento.
background-repeat: Controla como a imagem de fundo é repetida se for menor do que a área do elemento.
background-position: Determina a posição inicial da imagem de fundo.
background-size: Controla o tamanho da imagem de fundo.
background-origin: Define a origem do background em relação à caixa do elemento.
background-attachment: Determina se a imagem de fundo rola com o conteúdo ou permanece fixa
enquanto a página é rolada. background-blend-mode: Aplica um modo de mistura à imagem de fundo e à cor de fundo.
background: Propriedade abreviada que combina várias configurações do background em uma única declaração.
